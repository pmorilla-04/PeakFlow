# 🚴 PeakFlow (Work in Progress)

A full-stack SaaS platform for endurance athletes and coaches.

> **Status:** 🚧 First Version (MVP) - Under Development

## 📖 About the Project

This project is the first version of a sports training management platform inspired by tools such as TrainingPeaks.

The main goal is to create a complete web application that allows athletes to plan, analyze, and monitor their training sessions while learning modern web development technologies throughout the Higher Degree in Web Application Development (DAW).

The platform will continuously evolve by adding new features as new technologies are learned.

---

# 🎯 Project Goals

* Learn Full-Stack Web Development
* Practice Java and Spring Boot
* Design and manage relational databases with MySQL
* Build REST APIs
* Implement user authentication and authorization
* Visualize sports performance data
* Develop a real-world SaaS application

---

# 🚀 Current Version (MVP)

The first version focuses on the core functionality of the application.

### 👤 User Management

* User registration
* Secure login
* User profile

### 📅 Training Calendar

* View training sessions
* Create workouts
* Edit workouts
* Delete workouts

### 🚴 Training Activities

* Manual activity registration
* Workout details
* Sport type
* Duration
* Distance
* Elevation
* Notes

### 📝 Training Notes

Store personal information about every session:

* Feelings
* RPE (Rate of Perceived Exertion)
* Nutrition
* Equipment used
* Personal comments

### 📊 Dashboard

General statistics including:

* Total activities
* Total training hours
* Distance completed
* Elevation gained
* Recent activities

### 🔍 Filters

Filter activities by:

* Sport
* Date
* Training type
